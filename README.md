# SETRA Bölüm 5 — Dengeli Konsol Stabilite Hesabı

SETRA GB Chapter 5 / BPEL 91 / BAEL 91 esaslarına göre dengeli konsol köprü stabilite hesabı.

## Windows EXE İndirme

1. Sağ üstte **Actions** sekmesine tıklayın
2. **Windows EXE Derle** → en son çalışmayı açın
3. **Artifacts** bölümünden `SETRA_B5_Stabilite_Windows.zip` indirin
4. ZIP içindeki `SETRA_B5_Stabilite.exe` dosyasını çalıştırın

## Özellikler

- §5.3 Krawsky formülleri ile ağırlık hesabı
- §5.4 Tip A (Geçici) ve Tip B (Kazara) kombinasyonları
- §5.5.1 Tendon boyutlandırması
- §5.5.2 Blok yüzey alanı kontrolü (BAEL 91 A.8.4)
- §5.5.3 Ayak dönme açısı (≤1°)
- PDF rapor ve Excel çıktısı
